\subsubsection*{template$<$Numeric T$>$\newline
class maf::\+math::\+\+Vector$<$ T $>$}
A general-\/purpose mathematical vector class. 

This class implements a mathematical vector, wrapping a std::\+vector for data storage. It supports both {\bfseries{ROW}} and {\bfseries{COLUMN}} orientations, which is crucial for correct algebraic operations with the \doxylink{classmaf_1_1math_1_1_matrix}{Matrix} class.

It is templated to support various numeric types (T) and provides a range of constructors, element-\/wise operators, and common vector operations like norm, normalization, and dot product.
