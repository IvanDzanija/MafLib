\doxysubsection{Detailed Description}
High-\/performance PLU decomposition for square matrices. 

This header defines the {\ttfamily plu} function, which computes the PLU decomposition of a given square matrix A, where PA = LU. The implementation uses a blocked right-\/looking algorithm for improved cache performance and is parallelized with Open\+MP for multi-\/threading and SIMD vectorization.

The function checks for squareness and detects singularity during the computation. It supports both single and double precision floating point types, and allows for an optional template parameter to specify the result type.
