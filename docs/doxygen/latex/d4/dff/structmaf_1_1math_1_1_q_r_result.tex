This struct contains the orthogonal matrix Q and the upper triangular matrix R resulting from the QR decomposition. The types of Q and R are both \doxylink{classmaf_1_1math_1_1_matrix}{Matrix$<$\+T$>$}, where T is a floating point type. The user can choose to return either the full or thin versions of Q and R based on their needs when calling the QR\+\_\+decomposition function.
