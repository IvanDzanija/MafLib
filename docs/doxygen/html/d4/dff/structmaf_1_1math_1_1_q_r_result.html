Detailed Description

template<std::floating_point T>
struct maf::math::QRResult< T >

Struct to hold the results of QR decomposition.

Template Parameters
TThe floating point type of the matrix elements (e.g., float, double).

This struct contains the orthogonal matrix Q and the upper triangular matrix R resulting from the QR decomposition. The types of Q and R are both Matrix<T>, where T is a floating point type. The user can choose to return either the full or thin versions of Q and R based on their needs when calling the QR_decomposition function.
