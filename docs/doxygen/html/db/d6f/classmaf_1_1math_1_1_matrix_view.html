Detailed Description

template<Numeric T>
class maf::math::MatrixView< T >

A lightweight view into a submatrix of an existing Matrix. This class does not own the data; it provides a window into the data of another Matrix.

Template Parameters
TThe numeric type of the matrix elements (e.g., float, double).
