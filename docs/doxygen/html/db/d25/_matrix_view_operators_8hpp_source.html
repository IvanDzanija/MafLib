20 * It is usually worth casting a buffer for BLAS performance gains before hand.
21 *
22 * @tparam T The type of the Matrix-like object (MatrixView or Matrix).
23 * @tparam V The type of the Vector-like object (VectorView or Vector).
24 * @param A The Matrix-like operand.
25 * @param x The Vector-like operand.
26 * @return A new Vector resulting from the multiplication A * x.
27 */
