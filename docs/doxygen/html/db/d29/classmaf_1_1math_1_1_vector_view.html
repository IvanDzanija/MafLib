Detailed Description

template<Numeric T>
class maf::math::VectorView< T >

A lightweight view into a subvector of an existing Vector. This class does not own the data; it provides a window into the data of another Vector.

Template Parameters
TThe numeric type of the vector elements (e.g., float, double).
