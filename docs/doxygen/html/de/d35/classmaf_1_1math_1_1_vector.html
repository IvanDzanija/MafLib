Detailed Description

template<Numeric T>
class maf::math::Vector< T >

A general-purpose mathematical vector class.

This class implements a mathematical vector, wrapping a std::vector for data storage. It supports both ROW and COLUMN orientations, which is crucial for correct algebraic operations with the Matrix class.

It is templated to support various numeric types (T) and provides a range of constructors, element-wise operators, and common vector operations like norm, normalization, and dot product.

Template Parameters
TThe numeric type of the vector elements (e.g., float, double, int).
